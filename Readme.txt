No hay que instalar ningun componente, la integracion es facil y rapida
el componente App llamara el componente llamado instrucciones que contiene todo el codigo a usar
el componente App solo posee la constante images que es la que llama todas las imagenes para el tutorial
en Instrucciones en el metodo componentDidMount puedes controlar la velocidad que se mostrara las imagenes
del tutorial por defecto la deje en 3000.
para conocer en que slider se esta en el momento se usa selectedIndex
